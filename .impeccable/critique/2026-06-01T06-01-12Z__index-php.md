## Anti-Patterns Verdict

**LLM assessment:** ไม่ใช่ generic AI SaaS 100% — มี brand ส้ม TNC และ Sarabun ชัด แต่ยังมี tells: glassmorphism บน sidebar card, action buttons สีน้ำเงิน Bootstrap, stat cards แบบ template

**Deterministic scan:** `layout-transition` (height animation บน active bar), `single-font` (Sarabun only — ยอมรับได้ตาม DESIGN.md product register)

## Overall Impression

Dashboard ทำงานได้และข้อมูลครบ แต่ **brand drift (blue actions)** และ **sidebar ที่ปิดหมดตอนโหลด** ทำให้ office user ต้องจำว่ามีเมนูซ่อนอยู่ — ขัดกับเป้า workflow เร็วขึ้น

## What's Working

- Stat cards + invoice table อ่านยอดและสถานะได้เร็ว
- Orange active bar บน hub link สื่อสารชัด
- Skeleton loading ลด layout shift ตอนโหลด invoice

## Priority Issues

- **[P1] Blue action buttons ไม่ตรง DESIGN.md** — `.btn-invoice-action-view` ใช้ `#0d6efd`. **Fix:** เปลี่ยนเป็น orange-tinted หรือ neutral. **Command:** `/impeccable colorize index dashboard actions`
- **[P1] Sidebar เริ่ม collapsed ทั้งหมด** — `$index_hub_start_all_collapsed = true` ซ่อนเมนูหลัก. **Fix:** เปิดหมวด «ข้อมูลหลัก» เป็นค่าเริ่มต้น. **Command:** `/impeccable layout index sidebar`
- **[P2] Cognitive load สูงบน mobile** — hub ย้ายไป navbar slot ต้องเปิดเพิ่ม 1 ขั้น. **Command:** `/impeccable distill index mobile nav`
- **[P2] Glass blur หลายชั้น** — performance + visual noise. **Command:** `/impeccable quieter index dashboard cards`
